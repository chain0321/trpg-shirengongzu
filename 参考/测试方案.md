# 十人格测试 · 详细设计方案

## 一、定位与目标

**一句话定位**：通过 4 个虚构情境、24 道选择题，测出玩家内心"十人小组"的人格力量分布，为 TRPG 游戏匹配角色做准备。

**与传统测试的区别**：
- MBTI 给你贴一个标签 → 我们画一张力量地图
- 传统测试问"你觉得自己是不是…" → 我们让你在故事里做选择
- 传统结果是静态的 → 我们的结果包含"主导人格"和"被压抑的影子人格"

**测试时长**：约 8-12 分钟

---

## 二、10 个人格维度定义

测试需要量化的 10 个维度，每个维度对应一个核心驱动力：

| 编号 | 人格 | 核心驱动力 | 高分特征 | 低分特征 |
|------|------|----------|---------|---------|
| 1 | 法官 | 秩序与公正 | 重视规则、追求公平、害怕失控 | 灵活变通、不喜约束 |
| 2 | 教授 | 理性与求知 | 逻辑思考、追求真理、信仰知识 | 跟着感觉走、不爱分析 |
| 3 | 暴君 | 权力与执行 | 果断决策、敢做坏人、推动结果 | 温和退让、不争权力 |
| 4 | 小丑 | 破坏与创新 | 打破常规、制造意外、不按牌理出牌 | 稳重保守、不冒险 |
| 5 | 灵婆 | 直觉与联结 | 感知他人情绪、洞察深层联系、依赖直觉 | 理性导向、不信感觉 |
| 6 | X | 随机与混沌 | 享受不确定、制造变量、无目的行动 | 需要明确方向、讨厌意外 |
| 7 | 叛君 | 欲望与阴影 | 洞察他人隐藏面、挑战面具、揭示矛盾 | 维护表面和谐、回避冲突 |
| 8 | 少年 | 理想与反叛 | 坚持信念、对抗时代、不愿妥协 | 务实随俗、接受现状 |
| 9 | 摆渡人 | 慈悲与超脱 | 关注苦难、引导他人、保持距离的温柔 | 深度卷入、不愿放手 |
| 10 | 史官 | 记录与整合 | 观察全局、编织线索、寻找意义 | 活在当下、不回看过去 |

## 三、测试结构：4 个场景 × 6 道题

### 场景一：末日黎明（测试危机反应）
> 背景设定：你生活的城市突然陷入一场无法解释的灾难。通讯中断，天空被灰烬覆盖，街上只剩下零星的幸存者。你站在一栋半塌的建筑前，身后是你认识的几个人。

**Q1 — 灾难降临的第一反应**
你听到远处的爆炸声越来越近，你的第一个念头是：
- A）我们需要马上建立规矩，谁负责什么，不能乱 → 法官+2，暴君+1
- B）等等，先搞清楚到底发生了什么，有没有规律 → 教授+2，史官+1
- C）管它呢，能活就行，先跑再说 → X+2，小丑+1
- D）我感觉……那边不对劲，我们应该往反方向走 → 灵婆+2，摆渡人+1
- E）看看大家的状态，有人受伤了吗？ → 摆渡人+2，灵婆+1

**Q2 — 一个受伤的陌生人**
路上遇到一个受伤的陌生人，他请求加入你们。但你注意到他在隐瞒什么——他的包里似乎有不属于他的东西。你会：
- A）可以加入，但先说清楚规矩，违反就走人 → 法官+2，暴君+1
- B）质问他包里是什么，搞清楚再决定 → 叛君+2，教授+1
- C）先让他加入吧，人多力量大，观察着就行 → 摆渡人+2，史官+1
- D）不管了，反正末日了，一切重新开始 → 少年+2，小丑+1

**Q3 — 物资分配**
你们找到了一个物资充足的避难所，但只够 8 个人用，而你们有 12 个人。你倾向于：
- A）按贡献和能力来分配，必须有标准 → 法官+2，教授+1
- B）我来拿主意，有人必须做出牺牲 → 暴君+2，法官+1
- C）抽签决定，这样最公平 → X+2，小丑+1
- D）想办法扩容，或者去找新的避难所 → 少年+2，教授+1
- E）先照顾最虚弱的人，其他的再说 → 摆渡人+2，灵婆+1

**Q4 — 有人要离队**
一个同伴说他想独自出去找家人，你明知外面很危险。你会：
- A）不行，我们定过规矩，现在不能单独行动 → 法官+2，暴君+1
- B）理解，但先等我们制定一个搜救计划 → 教授+2，史官+1
- C）我陪你一起去 → 少年+2，摆渡人+1
- D）你自己想清楚，我尊重你的选择 → 摆渡人+2，X+1

**Q5 — 夜晚对话**
夜里大家围着火堆，有人问："你觉得这一切为什么会发生？"你更倾向于说：
- A）肯定是有人或有什么制度失败了，可以分析出来 → 教授+2，法官+1
- B）也许这就是命运，有些事注定要发生 → 灵婆+2，摆渡人+1
- C）管它为什么，重要的是我们现在怎么办 → 暴君+2，少年+1
- D）说不定是个好事呢，旧世界本来就该结束了 → 小丑+2，叛君+1
- E）我更想知道的是：我们能从中学到什么 → 史官+2，教授+1

**Q6 — 发现真相**
你发现这场灾难可能是某个组织故意制造的，你掌握了一部分证据。你会：
- A）收集完整证据，找到真相再公开 → 教授+2，史官+1
- B）立刻告诉大家，不管证据够不够 → 小丑+2，少年+1
- C）先告诉可以信任的人，一起决定怎么做 → 灵婆+2，法官+1
- D）利用这个信息来保护我们这群人 → 暴君+2，叛君+1

### 场景二：十人孤岛（测试团体角色）
> 背景设定：灾难过后，你和另外 9 个幸存者漂流到一座无人岛。你们需要建立一个新的小社会。岛上有淡水、野果和木材，但没有任何现代工具。

**Q7 — 第一天**
到达后的第一天，大家乱成一团。你会自然地去做什么？
- A）站出来组织分工，安排人去探查、找水、搭棚 → 暴君+2，法官+1
- B）先四处看看环境，画个地图，了解资源分布 → 教授+2，史官+1
- C）去和每个人聊聊，看看大家的状态和能力 → 灵婆+2，摆渡人+1
- D）找个高处坐下来，观察这些人，看谁可信 → 叛君+2，史官+1

**Q8 — 制定规则**
有人提议"我们需要定一套规则"，你的态度是：
- A）完全同意，而且我来起草 → 法官+3
- B）可以，但别太多，两三条核心的就够了 → 教授+1，少年+1，暴君+1
- C）规则会限制我们，不如看情况随机应变 → X+2，小丑+1
- D）先试着一起生活，有问题了再立规矩 → 灵婆+1，摆渡人+1，X+1

**Q9 — 人际冲突**
两个人因为食物分配吵起来了，越吵越凶。你会：
- A）站出来调停，基于之前定的规矩来裁决 → 法官+2，暴君+1
- B）让他们吵完，冲突暴露问题才好解决 → 叛君+2，小丑+1
- C）拉开他们，分别聊聊，了解各自的真实感受 → 灵婆+2，摆渡人+1
- D）提一个新方案，让两边都有台阶下 → 教授+1，史官+1，摆渡人+1

**Q10 — 有人不干活**
一个人每天什么都不做，只是坐在海边发呆。大家开始不满。你怎么看？
- A）不行，大家必须公平付出，要么干活要么走 → 法官+2，暴君+1
- B）也许他有自己的问题，先去了解一下 → 摆渡人+2，灵婆+1
- C）管他呢，也许他就是在用自己的方式存在 → X+2，少年+1
- D）观察他到底在想什么，也许他看到了我们没看到的 → 叛君+1，灵婆+1，史官+1

**Q11 — 一个秘密**
你无意中发现其中一个人在偷偷藏食物。你会：
- A）公开揭发，规矩就是规矩 → 法官+2，小丑+1
- B）私下跟他谈，问清楚原因 → 灵婆+2，摆渡人+1
- C）先不说，但暗中观察他还会做什么 → 叛君+2，史官+1
- D）直接拿回来分给大家，不用废话 → 暴君+2，少年+1

**Q12 — 岛上仪式**
生活稳定后，有人提议搞一个仪式或节日，来纪念你们的新开始。你会：
- A）好主意，我来记录我们到现在为止的故事 → 史官+2，灵婆+1
- B）可以，但别搞形式主义，有意义就行 → 少年+2，教授+1
- C）太好了！我来策划，搞点大家意想不到的 → 小丑+2，X+1
- D）没必要，先把生存问题解决好 → 暴君+2，法官+1

### 场景三：道德裂缝（测试价值观深层）
> 背景设定：你们的小社会已经运转了一段时间。但一些更深层的问题开始浮现——关于信任、牺牲、真相和权力。

**Q13 — 一个危险的选择**
你们发现岛的另一边有一个更大的幸存者群体。他们物资更多，但据说规矩很严厉，而且他们提出的条件是：合并，但必须服从他们的领导。你的立场是：
- A）可以谈判，但我们的核心规则不能放弃 → 法官+2，教授+1
- B）合并有利，为了大家生存可以暂时接受 → 暴君+2，教授+1
- C）绝对不行，我们好不容易建立了自己的方式 → 少年+2，法官+1
- D）先派人过去看看真实情况，再做决定 → 史官+2，叛君+1

**Q14 — 谎言的代价**
为了保护团队士气，有人建了一个善意的谎言（比如"外面的世界已经在恢复了"）。你发现了真相。你会：
- A）必须说出真相，人们有权知道 → 少年+2，法官+1
- B）理解他的用意，但谎言迟早会崩塌 → 教授+2，史官+1
- C）看情况，也许现在不是公布真相的好时机 → 灵婆+2，暴君+1
- D）有意思，谎言本身就说明了很多问题 → 叛君+2，小丑+1

**Q15 — 一个人的牺牲**
面临一个困境：需要有人冒极大风险去执行一个任务（比如穿过危险地带取药品），成功了全组人受益，失败了那个人可能回不来。你会：
- A）我去，有些事总要有人做 → 少年+2，暴君+1
- B）应该抽签或者自愿，不能强迫 → 法官+2，X+1
- C）先想想有没有其他方案，别急着牺牲 → 教授+2，灵婆+1
- D）看谁的能力最适合，理性选择 → 暴君+1，教授+1，史官+1

**Q16 — 权力的滋味**
因为你的贡献，大家推举你做决策人。拥有了"权力"后，你最可能：
- A）建立制度，让权力有章可循，避免独裁 → 法官+2，教授+1
- B）用权力快速推动重要的事，效率第一 → 暴君+2，教授+1
- C）不太舒服，我不想要权力，大家一起商量就好 → 摆渡人+2，少年+1
- D）有点兴奋，但也很警惕，权力会改变人 → 叛君+2，史官+1

**Q17 — 失去与记忆**
岛上有一个地方被大家当作"记忆角"，每个人在那里留下了一件旧世界的物品。你会留下什么类型的东西？
- A）一本书或笔记，知识不能丢 → 教授+2，史官+1
- B）一张照片或一封信，关于最重要的人 → 灵婆+2，摆渡人+1
- C）什么都不留，旧世界的东西该断就断 → 少年+2，小丑+1
- D）一个有象征意义的物件，代表一个还没完成的承诺 → 叛君+1，少年+1，史官+1

**Q18 — 最后的夜晚**
如果明天一切就结束了——不管是被救还是被毁灭——今晚你最想做什么？
- A）把所有发生过的事记录下来，不能被忘记 → 史官+3
- B）和每个人好好说说话，说那些平时没说的 → 灵婆+2，摆渡人+1
- C）一个人安静地待着，想想这一切意味着什么 → 教授+1，少年+1，摆渡人+1
- D）搞一场疯狂的庆祝，管它明天怎样 → 小丑+2，X+1

## 四、评分逻辑

### 4.1 原始分计算
- 每个选项为 1-3 个人格加分（权重 1-3 分）
- 24 道题做完后，得到 10 个人格的原始分
- 每个人格的理论最高分约为 15-18 分

### 4.2 标准化
- 将 10 个人格的原始分归一化为百分比（总和 = 100%）
- 或按每个人格的理论最大值计算百分比

### 4.3 分层
- **主导人格**：得分最高的 1-2 个（占比 > 15%）
- **活跃人格**：得分中等的 3-4 个（占比 8%-15%）
- **沉睡人格**：得分较低的 2-3 个（占比 < 8%）
- **影子人格**：得分最低的 1 个 + Q23（面具题）的加权

### 4.4 影子人格的特殊处理
Q23 是"反向题"——玩家选的不是自己强的那面，而是自己压抑的那面。这道题的得分单独计入"影子人格"标识，与最低分人格交叉验证。

### 5.2 人格描述文案风格
不用心理学术语，用你们手册的**叙事体**风格。举例：

> **法官主导**
> 你的心里住着一位穿黑袍的人。在所有人都在慌张、争吵、逃跑的时候，他会站起来，像一面旗帜。他相信：只要有规则，混乱就不会赢。但他也知道，规则是给活人定的——如果规则让人活不下去，那规则本身就成了暴政。这是他永恒的课题。

> **小丑作为影子**
> 你最不愿意承认的那面，是一个想把桌子掀翻的声音。他觉得所有正经的东西都有点可笑，所有严肃的面孔背后都藏着荒诞。你压制他，因为你觉得世界需要秩序，不需要混乱。但也许，他只是想告诉你：别太累了。

## 六、技术方案建议

### 6.1 技术选型
- **前端**：纯静态页面（HTML + CSS + JS），无需后端
- **样式**：与你们十人小组的视觉风格一致（红+黑+白）
- **图表**：用 Canvas 或 SVG 画雷达图
- **部署**：可以直接放在任何静态托管上

### 6.2 页面流程
```
引导页（介绍测试） 
  → 场景一（6题，带场景插画和渐进叙事） 
  → 场景二（6题） 
  → 场景三（6题） 
  → 场景四（6题） 
  → 计算中（动画过渡） 
  → 结果页
```

### 6.3 交互细节
- 每个场景开头有一段**沉浸式的场景描述**（3-4句话），配合氛围视觉
- 选项不标 ABCDE，用**自然语言卡片**的方式呈现
- 选完一题后**平滑过渡**到下一题，不要翻页感
- 场景切换时用**转场动画**（比如画面渐暗→新场景渐亮）
